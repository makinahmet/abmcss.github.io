The select element defines a drop-down list. You can adjust it with standart classes. To change the hover color of options you need to implement it yourself because it is rendered by OS.
The datalist element specifies a list of pre-defined options for an input element. Users will see a drop-down list of the pre-defined options as they input data. The list attribute of the input element, must refer to the id attribute of the datalist element.

Like select elements, the datalist element has very little flexibility in styling. Browsers define their own styles for these elements.
