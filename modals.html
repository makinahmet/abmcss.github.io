MODALS
   Modals are very common way to show users some information. Sometimes this can be images. ABMCSS uses hidden checkbox to show user modal. So javascript is not necessary for now. Also ABMCSS supports multiple modals in one page. Nested modals are supported too.
   To create a modal, you need labels and hidden chekbox. A modal have modal trigger, modal background and modal inner for contents. You can style modal trigger as ypu wish. For modal inner min widht is 300 px and min height is 100 px by default. If your content is bigger than this dimensions, modal inner will fit itself.
