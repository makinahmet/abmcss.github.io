TOOLTIPS
Tooltips are very useful to inform users when they hover to an element. In ABMCSS tooltips have standart 200 px width because positioning the tooltip element as absolute.

To have an tooltip you need to assign tooltip class to parent element and assign tooltipTop or tooltipRight or tooltipLeft or tooltipBottom to child element. You can still change all others properties of tooltip except its width.
When using tooltips, becareful on mobile devices. Just use top tooltips and bottom tooltips. If you use left or right tooltips, probably they wont appear to user.
